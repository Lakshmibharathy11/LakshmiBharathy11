## Featured Project

**[Marketing Mix Modeling — Budget Allocator](https://github.com/Lakshmibharathy11/Marketing-Mix-Modeling-for-Multichannel-Budget-Allocation)**
Bayesian MMM using Google Meridian across 5 channels on 156 weeks of geo-level data. VIF-informed priors handled collinearity where data alone couldn't isolate individual channel effects. Recommended a $4.8M budget reallocation, projecting a 0.5% conversion lift, delivered through a Looker Studio dashboard.
`Python` `Google Meridian` `MCMC` `Looker Studio`
